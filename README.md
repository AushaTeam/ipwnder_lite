# ipwnder_lite  
lightweight version of ipwnder  


## Support  
| chip | name |
|---------|----------|
| S5L8950 | Apple A6 |
| S5L8960 | Apple A7 |
| T8010 | Apple A10 |


## How to use?
- for iOS 14 environment, binary must be placed under `/usr/local/bin`.  

### cmd
```
ipwnder_lite [-p/-d]  
```
- `-p`: enter pwndfu  
- `-d`: demote and enable jtag/swd
